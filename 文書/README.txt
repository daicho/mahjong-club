＜成績管理システム＞
〇使い方
基本的にマニュアルに記載

・New.batを起動→Aと入力
・Excelで記録　点数入力のマクロが消えたらExcel上で"表示"→"マクロ"→"マクロの表示"
・試合終了後は保存し、成績.xlsmを起動、集計ボタン（左上）を押す

〇パソコン2台以上用いる場合

usbで成績管理システムのみコピーして移動（New.batにBと入力）
→"対局"フォルダの当日のExcelファイルのみA卓PCの"対局"フォルダに移動して集計

〇TortoiseGitの操作
 
・ダウンロード：Git Clone...　　　　　　　　　    →PCにダウンロードするとき
・更新(アップロード)：Git CommitからCommit&Push　 →毎回終わったら
・最新の状態にする：Git SyncからPull 　　　　　   →毎回始まる前
・集計：binからmjcm.exe                            →起動したら集計して更新される？（使ってない）

・Excelがバグったり、データミスして集計しちゃったりしたときはフォルダ消してGitHubで日付戻ってダウンロードして修正、更新し直す
